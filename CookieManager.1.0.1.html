This service can enumerate and remove cookies for a certain domain.

It can currently not add cookies, but this can be achieved through ECMAScript (document.cookie).

Changelog: 1.0 - First release 1.0.1 - Fixed a bug in RemoveCookie where it would remove all cookies, not just in the specified domain.

GetCookieSettings

Get global settings regarding the allowed number of cookies, and the max allowed length a cookie. 

command GetCookieSettings(Default)
{
    }
    returns (CookieSettings)
    {

    • The current settings for the cookie manager in the opera host, these settings are read-only and may vary depending on the platform or device used. 

    • required uint32 maxCookies = 1;

      The maximum number of cookies the host can store in total. If there is no room for new cookies the host will remove the least recently used cookies to get enough space. 

    • required uint32 maxCookiesPerDomain = 2;

      The maximum number of cookies allowed in one domain. If there is no room for new cookies the host will remove the least recently used cookies in the domain to get enough space. 

    • required uint32 maxCookieLength = 3;

      The maxium byte length of a single cookie, if the cookie exceeds this length it will be discarded. 

    }
    = 4;
