This service allows you to read, enumerate and write Opera's user prefs. Please note that when you set a pref, it is stored on disk, and its value will persist to the subsequent launches of the same profile.

GetPref

Get the value of a preference. If the preference could not be found, or it for some other reason is not readable, an error will occur.

It is possible to get the current (actual) value of the pref, or the default value of the pref. 

command GetPref(GetPrefArg)
{
  • required string section = 1;
  • required string key = 2;
  •  optional Mode mode = 3 [default = 1];
    {
    • CURRENT = 1;

      Get the current value of the pref. 

    • DEFAULT = 2;

      Get the default value of the pref (not necessarily the used value). 

    }
}
returns (PrefValue)
{
  • required string value = 1;
}
= 1;

ListPrefs

Enumerate all prefs. 

command ListPrefs(ListPrefsArg)
{
  • optional bool sort = 1 [default = false];

    Whether to sort the preferences alphabetically or not. 

  • optional string section = 2;

    List the keys in the specified section only. If not present, all keys in all sections will be listed. 

}
returns (PrefList)
{
  •  repeated Pref prefList = 1;
    {
    •  required Type type = 1;
      {
      • STRING = 1;
      • INTEGER = 2;
      • BOOLEAN = 3;
      • FILE = 4;
      • REQUIRED_FILE = 5;
      • DIRECTORY = 6;
      • COLOR = 7;
      }
    • optional string section = 2;

      Preferences are divided into groups called sections. This field indicates which section this preference is a member of.

      Example: *Developer Tools* 

    • optional string key = 3;

      This is the key, or name of the preference.

      Example: *Developer Tools URL* 

    • optional string value = 4;

      The value of the preference. For instance, for the preference *Developer Tools URL* in the *Developer Tools* section, the default value is https://dragonfly.opera.com/app/. 

    • optional bool enabled = 5;

      Whether setting is overriden by global ini. If this value is missing, it means global ini is not supported. 

    }
}
= 2;

SetPref

command SetPref(SetPrefArg)
{
  • required string section = 1;
  • required string key = 2;
  • required string value = 3;
}

    }
