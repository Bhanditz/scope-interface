DocumentManager provides a way to monitor, and interact with individual documents/frames in a document tree (whether they have an ECMASCript environment or not).

Example functionality includes loading URLs in a certain frame, and enumerating all the documents in a certain Window.

ID fields

  • Frame IDs represent the entity that contains the document. When a new URL is loaded in a frame, the document ID will change, but the frame ID will not. This makes it possible to monitor the document history for individual frames. (Please note that the document ID will not change when a frame is reloaded).
  • Document ID represent the actual document. This is recreated when a new URL is loaded in its parent frame.

External ID fields

Some IDs used in this service originate (and can be used with) other services.

  • Window IDs are the same in all services, and can be used interchangeably everywhere. See WindowManager for window-centric commands and events.
  • Resource IDs in this service are compatible with resource ID fields found in ResourceManager.

GetHighlights

command GetHighlights(GetHighlightsArg)
{
  • required uint32 windowID = 1;

    Get highlights in this window. 

  • optional uint32 frameID = 2;

    If set, search only in the specified frame. The frame must be part of the document tree of the previously specified window. 

}
returns (HighlightList)
{
  • repeated Highlight highlightList = 1;

    When searching for text in a page, we often get multiple hits. This is a list of all the current hits. 

    {
    • required uint32 frameID = 1;

      Which frame this highlight is in. 

    • optional Node node = 2;

      The first common ancestor that contains the entire highlight. 

      {
      • required string tag = 1;

        E.g. "a", "div". 

      • optional string id = 2;

        Present if there is an ID attribute set. 

      }
    }
}
= 7;

GetPageText

command GetPageText(GetPageTextArg)
{
  • required uint32 windowID = 1;

    Get the text in this window. 

  • optional uint32 frameID = 2;

    If set, get the text only for the specified frame. The frame must be part of the document tree of the previously specified window. 

}
returns (PageText)
{
  • repeated DocumentText documentTextList = 1;
    {
    • required uint32 frameID = 1;

      Which frame this document sits in. 

    • required uint32 documentID = 2;

      The document that contains this text. 

    • required string text = 3;

      The (plain) text of the document. 

    }
}
= 8;

ListDocuments

command ListDocuments(ListDocumentsArg)
{
  • optional uint32 windowID = 1;
}
returns (DocumentList)
{
  • repeated Document documentList = 1;
    {
    • required uint32 windowID = 1;
    • required uint32 frameID = 2;
    • optional uint32 documentID = 3;
    • optional uint32 parentFrameID = 4;

      The parent frame, present only if the frame is not the top level frame. 

    • optional uint32 parentDocumentID = 5;

      Present unless this document is the top-level document in the Window. 

    • optional string url = 6;

      The current URL of the document. 

    • optional uint32 resourceID = 7;

      The resource used to generate this document. 

    • optional string frameElementName = 8;

      The value of the name attribute of the HTMLFrameElement. If the frame does not have a name, or this document is not associated with a HTMLFrameElement, this will field will not be present. 

    • optional string frameElementID = 9;

      The value of the id attribute of the HTMLFrameElement. If the frame does not have an id, or this document is not associated with a HTMLFrameElement, this will field will not be present. 

    }
}
= 1;
